Дайджест профессий для школьников

Электронная книга о профессиях внефтегазвой отрасли для школьников. Книга используется в образовательном проекте «Petroschool».

Основная задача проекта — рассказать школьникам в простой форме о профессиях в нефтегазовой сфере и университетах, в которых можно получить образование для начала своей карьеры в этой отрасли промышленности.

Книга состоит из семи разделов по каждому сектору нефтегазовой промышленности: от геологоразведки до IT. Внутри разделов рассказывается об должностных обязанностях, необходимых компетенциях специалистов данной отрасли, приводится сравнительный список университетов с отзывами выпускников.

При выборе университета для абитуриента большое значение имеет его месторасположение, проходной балл, количество бюджетных мест, наличие военной кафедры и совместных образовательных программ с зарубежными вузами. Вся эта информация располагается на одной странице: таблица слева для удобного сравнения вузов по параметрам и карта справа для визуального представления распределения отраслевых вузов по территории страны.

В формате pdf электронная книга не может адаптироваться под маленькую ширину экрана устройства, как на html‑странице в вебе. Поэтому в файле было сделано две версии книги: для широкого экрана и для узкого, чтобы пользователь мог выбрать более удобный для него формат.
